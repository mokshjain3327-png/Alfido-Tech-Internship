About
This is my personal portfolio website that serves as a digital resume and project showcase. It was built as part of an internship task with the goal of presenting my academic background, technical skills, and hands-on projects to potential employers and collaborators.

Scroll-reveal animations (fade-in on scroll)
Mobile hamburger menu for smaller screens
Skill pills grouped by category
Project cards with tech tags and GitHub links
Education section with scores and year ranges
Certifications and extracurricular highlights
Working contact form with success feedback
Clean dark theme with sky-blue accent colors
Bootstrap 5 grid system for layout

🚀 Getting Started
Option 1 — Open Locally
No build tools or dependencies needed.
bash# Clone the repository
git clone https://github.com/mokshjain3327-png/portfolio.git

# Navigate into the folder
cd portfolio

# Open index.html in your browser
open index.html
Option 2 — Deploy on GitHub Pages

Push index.html to a GitHub repository
Go to Settings → Pages
Under Source, select Branch: main and folder / (root)
Click Save
Your portfolio will be live at https://your-username.github.io/repository-name


📁 Project Structure
portfolio/
│
└── index.html        # Single-file portfolio (HTML + CSS + JS combined)
The entire project is a single self-contained HTML file. All CSS is written in a <style> block in the <head>, and all JavaScript is in a <script> block at the bottom of the <body>. Bootstrap 5 and Google Fonts are loaded via CDN.
